# Restaurant Billing System

## Project Description

Restaurant Billing System is a console-based mini project developed in the C programming language. It helps restaurants automate the billing process by displaying the food menu, accepting customer orders, calculating the bill with GST, and printing a formatted invoice.

---

## Features

- Display Restaurant Menu
- Take Customer Orders
- Calculate Bill Automatically
- Apply 5% GST
- Generate Invoice
- Menu Driven Program
- Simple Console Interface
